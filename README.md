A light weight coding template.
Combined pure, foundation, font-awesome and others.
Use <a href="http://glenmaddern.com/articles/introducing-am-css">Attribute Module</a> to rewrite.
<h3>Structure</h3>
<pre>
assets -- scss -- animate
               -- bourbon
               -- font-awesome
               -- normalize
               -- owlCarousel
               -- pure -- button.scss
                       -- form.scss
                       -- function.scss
                       -- grid.scss
                       -- table.scss
                       -- visibility.scss
               -- site -- setting.scss
                       -- module.scss
                       -- ...
                       -- mq.scss
       -- js
       -- fonts
       -- css
</pre>
